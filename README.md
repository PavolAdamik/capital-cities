# 🌍 World Capital Cities Quiz (Multiplayer)

An interactive, web-based geography quiz designed for competitive learning. Test your knowledge by guessing capital cities from different continents in a sleek, dark-themed interface.

---

## 🚀 Live Demo
You can run this quiz by simply opening the `index.html` file in any modern web browser (Chrome, Safari, Firefox).

## ✨ Features

* **👥 Multiplayer Support**: Play with 1 to 10 players with custom names.
* **⏱️ Dynamic Timer**: Each turn has a 50-second limit to keep the game fast-paced.
* **🌍 Continent Filtering**: Practice specific regions (Europe, Asia, Africa, Americas, Oceania) or take on the whole world.
* **🧠 Smart Feedback Logic**:
    * **Correct Answer**: 2-second delay before the next turn.
    * **Incorrect/Timeout**: 5-second delay to allow players to memorize the correct city.
* **⏸️ Pause Functionality**: Need a break? Pause the game and the timer anytime.
* **⌨️ PC Optimized**: Supports the `Enter` key for quick answer submission.
* **📱 Mobile Friendly**: Responsive design that works perfectly on smartphones and tablets.
* **🔤 Smart Validation**: Automatically ignores accents (diacritics) and case sensitivity (e.g., "Paříž", "pariz", and "PARIZ" are all accepted).

---

## 🛠️ Technical Details

The project is built using a "Single File" architecture for maximum portability:
* **HTML5**: Semantic structure.
* **CSS3**: Custom dark-themed UI with Flexbox and responsive media queries.
* **Vanilla JavaScript**: Pure logic without any external libraries or dependencies.
